# atom-certificate-view package

view x.509 certificate detail call your system `openssl`.

openssl version : OpenSSL 1.0.2j 26 Sep 2016.

Develop in MacOS, MicroSoft Windows not support.

## config

```cson
<!-- config.cson -->
*
  "certificate-view":
    display: true
    openssl: "/usr/local/bin/openssl"
```
